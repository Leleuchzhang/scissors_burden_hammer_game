用 C++ 语言实现一个C/S架构的多人在线“剪子包袱锤”游戏：
1. 传输层协议使用TCP

2. 服务器支持1000+同时在线

3. 客户端可用命令行形式，也可用GUI形式，不做限制

4. 客户端支持的玩家操作：
  a. 登录服务器。例如: login [username]
  b. 查看在线用户信息，包括用户名，当前状态，得分等。例如: info [all | username]
  c. 发起一个战局。例如: open [roundname]
  d. 加入一个战局。例如: join [roundname]
  e. 出招。例如: cast [scissor | paper | rock]
  f. 放弃。例如: quit
  g. 登出服务器。例如: logout
  h. 更多操作类型可以根据自己兴趣实现

5. 服务器支持的功能:
  a. 处理玩家登录/登出
  b. 维护玩家信息，包括用户名，当前状态，得分等
  c. 战局管理，包括开启，关闭，得分判定等

6. 扩展功能:
a. 基于linux实现客户端和服务器
b. 使用select/epoll等机制管理网络连接
c. 使用多进程，分别处理网络连接和游戏逻辑
d. 处理异常逻辑，比如网络中断，战局超时，登录唯一性等
e. 使用MySQL存储玩家信息
f. 使用多线程实现MySQL连接池
